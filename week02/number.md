# Number直接量正则
## 整数
```
const reg = /^-?/d+$/
```
## 浮点数
```
//.2这种or1.2这种
const reg = /^(-?./d+ |-?[1-9]./d+)$/
```
## 二进制数
```
const /^[01]+$/
```
## 八进制数
```
const reg = /^0[0-7]+$/
```
## 16进制数
```
const reg = /^0[xX][0-9A-Fa-f]+$/
```
## 匹配所有number直接量
```
const reg = /^(-?/d+|-?./d+|-?[1-9]./d+|[01]+|0[0-7]+|0[xX][0-9A-Fa-f]+ )$/
```
